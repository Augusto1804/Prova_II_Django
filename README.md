## Sobre o Projeto

Este projeto foi desenvolvido como parte da Prova de Desenvolvimento Web II, com base nas práticas realizadas em sala de aula utilizando o framework Django.

A proposta consiste na criação de um sistema simples de gestão de eventos, onde um administrador pode cadastrar eventos e visualizar uma lista com os eventos registrados.

---

## Gestão de Eventos

O projeto foi desenvolvido utilizando o VS Code e comandos simples para ativação do ambiente virtual (venv) e inicialização do projeto Django.

Basicamente, o sistema funciona da seguinte forma:

- O administrador registra um evento através de um formulário  
- Cada evento possui nome e localização  
- Após o cadastro, os eventos são exibidos em uma página de listagem  

---

## Tecnologias Utilizadas

- Python  
- Django  
- HTML  
- VS Code  
- Ambiente virtual (venv)

---

## Funcionalidades

- Cadastro de eventos  
- Registro de nome e localização  
- Listagem dos eventos cadastrados  
- Navegação entre páginas  

---

## Estrutura do Projeto

O projeto segue a estrutura básica do Django:

- **Templates:** páginas HTML  
- **Views:** lógica do sistema  
- **URLs:** rotas da aplicação  
- **Models (opcional):** estrutura dos dados
